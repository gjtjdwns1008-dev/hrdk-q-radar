# HRDK Q-RADAR — 지능형 법령 모니터링 시스템

> 매일 새벽, 대한민국의 제·개정 법령을 자동 수집하고 AI가 **국가기술자격과의 관련성·우대사항**을 분석하여
> 대국민 공개 사이트와 월간 브리핑으로 내보내는 한국산업인력공단 AI 프론티어 과제입니다.

**🔗 라이브 사이트: [자격증 법령 네비게이터](https://gjtjdwns1008-dev.github.io/hrdk-q-radar/)**

---

## 무엇을 하나요?

법령이 바뀌면 특정 자격증 소지자를 의무 채용해야 하는 일자리가 생기기도 하지만, 정작 그 자격증을 가진
구직자는 모르고 지나치기 쉽습니다. Q-RADAR는 이 "정보의 사각지대"를 메웁니다 — 사람이 읽기 힘든
분량의 법령을 AI가 매일 대신 읽고, **"어떤 자격증에, 어떤 이득(의무고용·직무권한·시험면제·인사우대)이
생겼는지"**를 누구나 볼 수 있는 화면으로 번역합니다.

## 라이브 화면 (3개 탭)

| 탭 | 내용 |
|---|---|
| **법령 모니터링 총괄현황** | 날짜별 수집·검토·우대 현황 한눈에 — 수치를 누르면 그날의 법령 목록과 상세 분석까지 |
| **자격증 활용도 모니터링** | 제·개정 법령별 노동시장 활용도 분석 (관련 종목·근거 조문 링크 포함) |
| **자격증별 우대사항** | 543개 자격증 카드 — 종목별 우대 법령과 분류(직능연 기준 4종) 조회 |

## 작동 구조

```
법제처 Open API ─┐
                 ├─→ GitHub Actions (매일 새벽, 30분 격자 재시도)
워크넷 API ──────┘         │
                           ▼
              Gemini AI 투트랙 분석
              · 별표(HWP/HWPX/PDF) 판독      · 환각 방지 가드
              · 정책 관점: 취급유형·위험도    · 구직자 관점: 효용코드
                           │
                           ▼
              Google Sheets 통합 대장 (마스터 DB)
                    │                │
                    ▼                ▼
        네비게이터 (GitHub Pages)   월간 이슈브리핑 (docx·xlsx 자동 발송)
```

수집 실패분(해외 IP 차단 등)은 정직하게 "검토필요"로 신고되고, `migrate_tool/`의
로컬 보완 루프가 이를 회수합니다 — **자가 치유형 파이프라인**입니다.

## 폴더 구조

```
pipeline/       일일 수집·분석 파이프라인 (main·brain·config·knowledge) + 브리핑 생성기
core/           공유 라이브러리 hrdk_law_core (수집·별표판독·워크넷·백필·DB)
navigator/      대국민 공개 사이트 빌더 (통합 대장 → 단일 index.html)
migrate_tool/   로컬 보완·재분석 도구 (검토필요 정리, 부분 재수집)
data/           SQLite 지식베이스 (직능연 우대법령 22,772행 등)
docs/           시스템 구조 설명서·변경 리포트
.github/        자동화 워크플로 (daily / build-navigator / run-briefing / run-manual)
```

## 자동화 스케줄

| 워크플로 | 주기 | 하는 일 |
|---|---|---|
| `daily` | 매일 새벽 (30분 격자) | 법령 수집→AI 분석→대장 기록. 성공 시 이후 슬롯은 몇 초 만에 조기 종료 |
| `build-navigator` | 매일 3회 | 통합 대장 → 공개 사이트 재빌드 (GitHub Pages) |
| `run-briefing` | 매월 2일 | 월간 이슈브리핑 docx·xlsx 생성 및 메일 발송 |
| `run-manual` | 수동 | 특정 일자 재수집 (MANUAL_DATE) |
