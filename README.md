# HRDK Q-RADAR — 지능형 법령 모니터링 (모노레포)

매일 제·개정되는 법령을 자동 수집하고 AI가 국가기술자격과의 관련성·우대사항을 분석하여
구글시트(통합 대장)에 축적, **월간 이슈브리핑**과 **대국민 공개 사이트(자격증 법령 네비게이터)**로
내보내는 한국산업인력공단 AI 프론티어 과제 시스템입니다.

| 위치 | 역할 | 실행 |
|---|---|---|
| 루트 `*.py` | 일일 배치 + 월간 브리핑 | daily.yml(30분 간격) / run-briefing.yml(매월 2일) |
| `core/` | 공유 라이브러리 hrdk_law_core | 워크플로우가 `pip install ./core` |
| `navigator/` | 공개 사이트 빌더 (GitHub Pages) | build-navigator.yml(하루 3회) |
| `migrate_tool/` | 이관·정비 도구 5종 | 수동·로컬 |
